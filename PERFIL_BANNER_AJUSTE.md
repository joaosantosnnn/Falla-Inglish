# Ajuste do banner diretamente no perfil

- Adicionado o botão **Ajustar capa** no banner ativo do perfil.
- O usuário pode arrastar a imagem com mouse, dedo ou caneta.
- Controle de zoom entre 100% e 240%.
- Botão Centralizar para restaurar o enquadramento.
- O ajuste é salvo separadamente para cada banner equipado.
- O mesmo enquadramento é aplicado na capa do perfil e no cartão de destaque da página inicial.
- Gradientes e banners CSS continuam funcionando normalmente; o editor aparece apenas para banners baseados em imagem.
