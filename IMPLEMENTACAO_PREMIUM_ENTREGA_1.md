# FALLA Premium — Entrega 1

Esta entrega cria a infraestrutura inicial do Premium:

- tabela de benefícios Premium;
- tabela de planos;
- tabela de assinaturas;
- serviço central `hasPremiumFeature(...)`;
- nova aba **Premium** no painel administrativo;
- ativação/desativação de benefícios;
- configuração de plano mensal e anual;
- fallback local caso o Supabase ainda não esteja configurado.

## Antes de usar o painel

Execute `SQL_PREMIUM_BASE.sql` no SQL Editor do Supabase.
