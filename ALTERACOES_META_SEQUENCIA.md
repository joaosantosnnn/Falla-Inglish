# Metas de sequência de acertos

- O efeito não aparece mais a cada acerto.
- O padrão agora é uma meta de 5 em 5: 5, 10, 15, 20...
- Um erro zera a sequência atual.
- A nova aba **Config. das Lições** no painel permite escolher uma meta entre 2 e 50.
- A configuração é salva no Supabase e mantida localmente como fallback.

Antes de usar o painel, execute `SQL_CONFIGURACAO_SEQUENCIA.sql` no SQL Editor do Supabase.
