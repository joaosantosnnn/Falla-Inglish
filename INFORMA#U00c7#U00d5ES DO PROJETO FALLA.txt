Sempre rode isso primeiro, antes de qualquer outro comando — é a pasta raiz do projeto.


🔧 Instalar/atualizar dependências

npm install
Rode sempre que baixar um zip novo do AI Studio ou o package.json mudar.


💻 Rodar localmente no navegador do PC (mais rápido pra testar)
npm run dev

Abre em http://localhost:3000 (ou porta similar). Ótimo para testar rápido sem precisar de Android Studio.


🏗️ Build para produção (gera os arquivos finais)
npm run build


📱 Sincronizar com o Android
npm run cap:sync

🤖 Abrir no Android Studio
npm run cap:open-android
