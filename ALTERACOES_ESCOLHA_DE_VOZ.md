# Escolha de voz no primeiro acesso

- Adicionada uma tela automática após o primeiro login para detectar as vozes gratuitas disponíveis no aparelho.
- O usuário pode ouvir uma amostra de cada voz antes de escolher.
- A voz escolhida é usada pelo serviço de leitura de perguntas e pronúncias.
- Incluídos controles de velocidade e tom.
- Incluído botão para atualizar a lista quando novas vozes forem ativadas no celular.
- Incluída a opção de continuar usando a voz padrão do aparelho.
- Adicionado o atalho “Voz das lições” em “Mais”, permitindo alterar a escolha depois.
- Nenhum serviço pago ou API externa foi adicionado.
